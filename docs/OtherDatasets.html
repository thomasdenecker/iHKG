Dataset 1

Analysis

Differential analyses were performed comparing 0.5, 1, 2 or 4 h to the 0 h time point. Note that this is indirect comparisons since the microarray were done with a common reference (see the Experiment description). The tool GEO2R was applied to generate R code with LIMMA method. Note that "singleton" genes were deleted before data import (as described on the Broad Institute website).

Experiment

In this study, the authors performed transcriptome analyses of the WT strain (C. glabrata) using microarrays. They compared the compositions of transcriptomes at different time points (0.5 h, 1h, 2h, 4h) after iron depletion, to 0 h. The cell growth conditions are described as followed (in the article) : "Cells were adjusted to 10^7 cells/ml in iron-free citrate-buffered SD (pH 5.8) supplemented with 5 microM FeCl3 and grown.

Data source

Gerwien F, Safyan A, Wisgott S, et al. A Novel Hybrid Iron Regulation Network Combines Features from Pathogenic and Nonpathogenic Yeasts. mBio. 2016;7(5):e01782-16. Published 2016 Oct 18. doi: 10.1128/mBio.01782-16

Dataset 2

Analysis

Protein abundances obtained in two cell growth conditions (alkaline pH or standard) were compared, in order to identify differentially expressed proteins. LIMMA method was applied with default parameters, in order to calculate p-values.

Experiment

In these experiments, mass spectrometry analyses were performed in the yeast Candida glabrata. Proteins were extracted using FASP protocol (by Camille Garcia from the platform proteomics@IJM). Technical and biolocal replicates were done in order to evaluate the variability associated to each type of data reproduction.

Data source

Lelandais G, Denecker T, Garcia C, Danila N, Léger T, Camadro JM. Label-free quantitative proteomics in Candida yeast species: technical and biological replicates to assess data reproducibility. BMC Res Notes. 2019;12(1):470. Published 2019 Aug 1. doi:10.1186/s13104-019-4505-8

Dataset 3

Analysis

Differential expression measurements together with statistical p-values were obtained from the Supplementary Table S1 in Linde et al. article. In this file we considered information in the spreadsheet named "Expression_pH_change". Note that logFC represent the ratio between pH4 and pH8 conditions, i.e. log(pH4/pH8). This values were modified to rather represent log(pH8/pH4).

Experiment

RNAseq experiments were performed to identify differentially expressed genes during pH shift in the pathogenic yeast Candida glabrata. C. glabrata was cultured by two subsequent overnight cultures in M199 pH4 at 37°C. 10^6 cells/mL were suspended in either M199 pH4 or pH8 for 60 min, at 37°C while shaking. More information can be found in Linde et al. article. Primary datasets can be downloaded from GEO under accession number GSE61606.

Data source

Linde J, Duggan S, Weber M, et al. Defining the transcriptomic landscape of Candida glabrata by RNA-Seq. Nucleic Acids Res. 2015;43(3):1392–1406. doi: 10.1093/nar/gku1357
